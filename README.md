## Installing OpenGL (IntelliJ)
-   Copy the files from 'lib' folder in project directory
-   Then goto File > Project Structure > Libraries
-   In 'Libraries' click ' + ' and select the 'lib'
-   Then press 'Apply' and 'Ok'
